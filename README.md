# SmartShop - Advanced E-Commerce SQL Project

This project simulates an online shopping database system.

## Features
- Users
- Products
- Categories
- Cart
- Orders
- Payments
- Trigger for auto stock update

## SQL Concepts Used
- Joins
- Foreign Keys
- Aggregate Functions
- Triggers
- Constraints

## How to Run
Run:
1. schema.sql
2. data.sql
3. queries.sql
